Eventastic Mobile Application & Admin Website

Eventastic application is an android application which has the capabilities of to suggest the user to insert important details about their event by segment and display all needed calculation and details by dashboard. This application used Google Map API with integration of MYSQL database to save data of the user. There is also a web-based admin system connected to this app for the admin to add/remove venues from the app. Developed over 40 test plan and test cases and drafted testing report for this project. This was made using JAVA, Android Studio, MYSQL, PHP, HTML, CSS, and Javascript .
